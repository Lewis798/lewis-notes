体验优化 ​

  • 自定义你的新标签页

  • Saladict 沙拉查词 一款专业划词翻译扩展，为交叉阅读而生。涵盖中英日韩法德西语，支持复杂的划词操作、网页翻译、生词本与 PDF 浏览
  • 彩云小译 全文翻译(中英对照)
  • 划词翻译 划词翻译
